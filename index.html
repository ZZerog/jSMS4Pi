The jSMS4Pi

jSMS4Pi is simple Java library which provides SMS API for USB 3G modems. Provide abstraction for sent / received sms and other features. Communication with modem is done through AT commands. jSMS4Pi is ideal for a small stand alone computers (as RaspberryPI) without internet connections. If you want to send SMS through internet try any of Bulksms service.

How it works

USB 3G modem creates a virtual serial port after connecting to computer. jSMS4Pi sends an/the AT command through serial port to the modem and reads the response.

Supported modems

jSMS4Pi supports all modems which use (virtual) serial port (USB 3G Modems, real serial port modems, bluetooth modems, cell phone, ...). Some modems implement incomplete AT command set or does not comply specification. I believe the set of tested modem devices will be growing.

Supported OS

jSMS4Pi should support all OS with a/the serial port and Java library for it. Now jSMS4Pi is using JSSC as a/the serial port and supports Win32(Win98-Win8), Win64, Linux(x86, x86-64, ARM), Solaris(x86, x86-64), Mac OS X 10.5 and higher(x86, x86-64, PPC, PPC64). See JSSC homepage. But so far is only tested for Linux OS.
