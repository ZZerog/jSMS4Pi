Debugging

If you have a problem with a Modem and you can see communication between jSMS4Pi and the Modem, you must enable logging. jSMS4Pi use Log4j 2, so the easiest way is to add a http://github.com/ZZerog/jSMS4Pi/blob/master/log4j2.xml file into the classpath. Sample http://github.com/ZZerog/jSMS4Pi/blob/master/log4j2.xml on GitHub generated jSMS4Pi.log file similar this:
